# Rasterizer-and-Shader
Rasterizer and Shader created for Computer Graphics assignment. 

## Running the code 
Using cmake, first create a `build` folder inside the `src` folder. Then, run 
```
cmake ../
```
For a simple rasterizer and shader, use 
```
make task2
```
and 
```
./task2
```
For shadows, use 
```
make task3
```
and 
```
./task3
```
It is noted that the shadow has not been created as realistically as it should.
